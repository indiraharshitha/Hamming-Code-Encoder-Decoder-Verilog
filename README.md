# Hamming-Code-Encoder-Decoder-Verilog

## Project Overview

This project implements a Hamming (7,4) Error Correction Encoder and Decoder using Verilog HDL in Xilinx Vivado.

The encoder converts 4-bit input data into a 7-bit Hamming code by generating parity bits.

The decoder detects and corrects single-bit errors using syndrome calculation.
